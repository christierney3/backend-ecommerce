## Installation: 
You will need to run nmp i, login to mysql and source db/schema.sql, run npm run seed, and finally nmp start. 
```bash
npm i
```

```bash
mysql> source db/schema.sql
```

```bash
npm run seed
```

```bash
npm start
```
